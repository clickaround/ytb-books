# Contributing

Thanks for your interest in this Claude Code skill pack.

## How to help

- ⭐ **Star this repo** — it helps others discover the project.
- 🐛 **Report bugs** via [Issues](../../issues).
- 💬 **Ideas, feedback, or questions** — open a [Discussion](../../discussions).
- 🔧 **Pull requests welcome** — small focused PRs preferred.

## Local development

1. Clone this repo
2. Read each `skills/<name>/SKILL.md`
3. Test with `claude` CLI: `/plugin install <name>`
